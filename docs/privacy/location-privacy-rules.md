# Location Privacy Rules

Wellnest location rules:

- Customer address is used for booking, navigation, ETA, support, and safety.
- Provider live location is visible only to the customer of the active booking.
- Provider location sharing starts after provider accepts a job.
- Provider location sharing stops after service starts, completes, cancels, or expires.
- Admin exact location access requires a reason and creates an audit log.
- Raw provider location events are retained for 72 hours by default.
- Analytics must not use raw identifiable movement history.
