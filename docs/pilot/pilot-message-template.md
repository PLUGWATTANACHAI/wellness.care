# Wellnest Pilot Invitation Message

สวัสดีครับ ตอนนี้พี่กำลังทดลองแอพ Wellnest สำหรับเรียกบริการนวด/Wellness ไปให้บริการที่คอนโด อยากชวนช่วยทดลองใช้รอบเล็กก่อนเปิดจริงครับ

สิ่งที่อยากให้ลอง:

1. เปิดแอพ
2. เลือกบริการ
3. เลือกวัน เวลา และสถานที่
4. ตรวจสอบหน้าสรุปก่อนจอง
5. ดูหน้าติดตามผู้ให้บริการ
6. ส่ง feedback ให้พี่ว่าตรงไหนงง ตรงไหนดี หรือตรงไหนยังไม่น่าไว้ใจ

Android:
ใช้ลิงก์ APK นี้

หมายเหตุ:
รอบนี้เป็น pilot กลุ่มเล็ก ยังไม่ใช่แอพบน Store จริง กรุณาอย่าแชร์ลิงก์ต่อ และยังไม่ต้องใส่ข้อมูลส่วนตัวที่ละเอียดเกินจำเป็นครับ
