# Wellnest Pilot Tester Instructions

## Android Testers

1. Open the APK link from the pilot invitation.
2. Download the file.
3. If Android asks for permission to install from browser, allow it only for this install.
4. Open Wellnest.
5. Login with your phone number and the closed tester OTP code from the operator.
6. Test these actions:
   - Open Account and save condo/address details before booking.
   - Choose a service.
   - Pick date and time.
   - Review booking summary.
   - Confirm the pilot booking.
   - Check provider tracking preview.
7. Send feedback using the feedback questions.

## iPhone Testers

1. Install TestFlight from the App Store if it is not already installed.
2. Open the TestFlight invitation from Apple.
3. Accept the Wellnest test invite.
4. Install Wellnest from TestFlight.
5. Login with your phone number and the closed tester OTP code from the operator.
6. Test these actions:
   - Open Account and save condo/address details before booking.
   - Choose a service.
   - Pick date and time.
   - Review booking summary.
   - Confirm the pilot booking.
   - Check provider tracking preview.
7. Send feedback using the feedback questions.

## Tester Rules

- Do not share the pilot link publicly.
- Do not share the closed tester OTP code outside the invited tester group.
- Do not enter sensitive personal information beyond what is needed for testing.
- Do not use real payment unless the owner confirms it is a live test.
- If location tracking wording feels unclear, report it immediately.
- If anything looks unsafe or confusing, stop and report the step.
