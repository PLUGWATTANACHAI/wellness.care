# Wellnest Pilot Tester Instructions

## Android Testers

1. Open the APK link from the pilot invitation.
2. Download the file.
3. If Android asks for permission to install from browser, allow it only for this install.
4. Open Wellnest.
5. Test these actions:
   - Choose a service.
   - Pick date and time.
   - Enter condo/address details.
   - Review booking summary.
   - Confirm the pilot booking.
   - Check provider tracking preview.
6. Send feedback using the feedback questions.

## iPhone Testers

1. Open the Wellnest Pilot public web link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Open Wellnest from the home screen icon.
5. Test these actions:
   - Choose a service.
   - Pick date and time.
   - Enter condo/address details.
   - Review booking summary.
   - Confirm the pilot booking.
   - Check provider tracking preview.
6. Send feedback using the feedback questions.

## Tester Rules

- Do not share the pilot link publicly.
- Do not enter sensitive personal information beyond what is needed for testing.
- Do not use real payment unless the owner confirms it is a live test.
- If location tracking wording feels unclear, report it immediately.
- If anything looks unsafe or confusing, stop and report the step.
